# Endpoints

## Example Endpoint

### `GET /api/v1/resource`

Returns a list of resources.

**Parameters:**

| Name   | Type   | Required | Description          |
|--------|--------|----------|----------------------|
| limit  | int    | No       | Max results (default 20) |
| offset | int    | No       | Pagination offset    |

**Response:**

```json
{
  "data": [],
  "total": 0,
  "limit": 20,
  "offset": 0
}
```
